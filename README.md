About

This repository contains game mechanics, C# scripts, and gameplay systems developed for Detective Game Ultimate using Unity.

It serves as a collection of reusable gameplay features, prototypes, and systems used during the game's development.

Repository Contents

- 🎮 Gameplay mechanics
- 📜 C# scripts
- 🧩 Interaction systems
- 🤖 AI logic
- 🔦 Flashlight mechanics
- 📖 Notes and collectible systems
- 💊 Gameplay items
- ⚙️ Various utility systems

«Note: This repository does not contain the full source code of the game. It includes selected mechanics and scripts from the Detective Game Ultimate project.»
